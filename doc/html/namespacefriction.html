Parameters
[in]muThe friction coefficient.
[in]normalThe normal force.
[in]epsA regularization parameter, typically quite small.
[in]velocityThe relative velocity of the two contacting bodies.
Returns
The resulting friction force.
Remarks
This model utilizes a regularization parameter \( \epsilon \) to assist in dealing with the discontinuity of the pure Coulomb model when velocity is zero. The model can be described as follows. \( F = -\frac{\mu N}{\epsilon} v \) when \( |v| \leq \epsilon \), and \( F = -\mu N sgn(v) \) when \( |v| > \epsilon \).
