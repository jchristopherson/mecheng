Splits a string by a delimiter character.

Parameters
[in]txtThe string to split.
[in]delimThe delimiter character.
[out]bufferA string buffer used as a container for each string. Each string in the collection will be allocated to a default size defined by def if not already that size. It is recommended to use trim or len_trim when using each string.
[in]defAn optional parameter specifying the size of each string within buffer. The default, if not supplied, is 1024.
[out]iworkAn optional workspace array of same size as buffer.
Returns
The actual number of strings written to buffer.
Example
program example
use iso_fortran_env
use strings
implicit none
! Create a string to split
character(len = *), parameter :: str = "This,is,a,test,,string,with,repeating,,,delimiters"
! Local Variables
type(string) :: buffer(50) ! Arbitrarily chosen buffer size
integer(int32) :: i, n
! Split the string
n = split_string(str, ",", buffer)
! Print the results
print '(A)', "Split: " // str
print '(AI0A)', " ", n, " strings created:"
do i = 1, n
print '(AI0A)', achar(9), i, ". " // trim(buffer(i)%str)
end do
end program

Parameters
[in]txtThe string to split.
[in]delimAn optional input that defines the delimiter character. The default is a comma.
[in]nbuffAn optional input that allows control over the size of the buffer that contains the entire list of items. The default is 10,000 items.
[in]buffsizeAn optional input that allows control over the size of each item buffer. The default is 4096 characters per item.
Returns
A collection of strings. The delimiter character is not returned.
