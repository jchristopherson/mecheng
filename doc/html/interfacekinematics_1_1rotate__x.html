Detailed Description

Creates a quaternion representing the specified rotation angle about the specified rotation axis.

Parameters
[in,out]thisThe quaternion object.
[in]angleThe rotation angle, in radians.
[in]axisA 3-element vector containing the x, y, and z components of the rotation axis. The vector is expected to be a unit vector. Converts the quaternion to its equivalent angle-axis representation.
[in]thisThe quaternion object.
[out]angleThe equivalent rotation angle, in radians.
[out]axisA 3-element array that on output contains the equivalent axis of rotation. Creates a quaternion equivalent to the supplied rotation matrix.
[in,out]thisThe quaternion object.
[in]xThe 3-by-3 rotation matrix. It is assumed that this is a valid, orthogonal rotation matrix. Matrix orthogonality is not verified. Normalizes the quaternion to a unit quaternion.
[in,out]thisThe quaternion object. Computes the magnitude of the quaternion.
[in]thisThe quaternion object.
