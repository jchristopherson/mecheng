Detailed Description

A type used to assist in training a neural network.

Remarks
This type provides a mechanism for efficient evaluation of the cost function and its gradient vector with respect to the network output. This particular type utilizes a quadratic cost function defined as follows. \( C = \frac{1}{2n} \sum\nolimits_{x} \left( y \left(x\right) - a \right)^{2} \). The gradient of this function is then as follows. \( \nabla_{a}C = \frac{a - y \left(x\right)}{n} \).
