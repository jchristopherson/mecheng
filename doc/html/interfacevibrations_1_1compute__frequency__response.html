Compute the frequency response of a linear vibrating system.

Parameters
[in]mAn N-by-N mass matrix.
[in]kAn N-by-N stiffness matrix.
[in]bAn N-by-N damping matrix.
[in]fAn M-by-N forcing function matrix with each of the M rows representing a discrete excitation frequency.
[in]freqAn M-element array containing the frequency points at which to analyze the system. The units are expected to be rad/s.
[in,out]errAn optional errors-based object that if provided can be used to retrieve information relating to any errors encountered during execution. If not provided, a default implementation of the errors class is used internally to provide error handling. Possible errors and warning messages that may be encountered are as follows.
  • MECH_ARRAY_SIZE_ERROR: Occurs if any of the input matrices are not square, or if the matrices are sized differently.
  • MECH_OUT_OF_MEMORY_ERROR: Occurs if there is insufficient memory available.
  • LA_CONVERGENCE_ERROR: Occurs if the eigen solver fails to converge.
Returns
An M-by-N element matrix where each of the N results can be written for all M frequency points of interest.
